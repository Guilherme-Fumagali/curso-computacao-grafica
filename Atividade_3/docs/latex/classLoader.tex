\doxysubsection{Detailed Description}
Class for loading .obj files. 

The class contains a static method for loading .obj files. The method takes a file path as argument and returns a vector of vertices and a vector of triangles. The vertices are stored as \{\begin{DoxySeeAlso}{See also}
vec3\} vectors and the triangles are stored as \{

\doxylink{classTriangle}{Triangle}\} objects. @\+Note In \doxylink{classTriangle}{Triangle}, if the index value is -\/1, it means that the value is not defined. 
\end{DoxySeeAlso}
