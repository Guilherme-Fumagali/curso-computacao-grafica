Esse projeto fornece uma classe para leitura de modelos .obj para C++.

Essa classe, chamada de \href{src/Loader.h}{\texttt{ Loader}}, recebe o caminho para um arquivo .obj e ponteiros de vetores, que serão escritos com os dados de vértices, normais, índices do modelo e informações das faces dos triângulos.

Para armazenar os dados das faces, foi criada a classe \href{src/Triangle.h}{\texttt{ Triangle}}, que armazena os índices dos vértices e normais de cada face.
